Semântica HTML5

A importância da semântica HTML5.

HTML semântico tem como objetivo descrever o significado do
conteúdo presente em documento HTML, tornando-o mais claro
tanto para programadores quanto para browsers.

Elementos Semânticos

Os elementos semânticos dão significados aos conteúdos.

Elemento Section

Esse elemento define uma seção em um documento HTML. de
acordo com a W3C: Uma seção é um agrupamento temático de
conteúdo, normamente com um título.

Fonte: https://www.w3schools.com

World Wide Web Consortium

O World Wide Web Consortium (W3C) é uma organização brinternacional não
lucrativa que estabelece padrões técnicos para a World Wide Web. Fundado
em 1994 por Tim Berners-Lee, o W3C é composto por empresas, órgãos
governamentais e organizações independentes que colaboram para o
desenvolvimento de tecnologias e padrões que garantam a evolução da web.

Elemento Article

Define algum conteúdo além daquele em que é colocado
- como barra lateral. O conteúdo deve estar
indiretamente relacionado ao conteúdo circindante.
