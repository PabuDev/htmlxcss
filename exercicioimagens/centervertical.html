Centralização Vertical

Este exemplo demonstra como centralizar verticalmente um conteúdo dentro de uma página HTML. A centralização é feita utilizando CSS Flexbox, que é uma técnica moderna e eficiente para layout.

O conteúdo é colocado dentro de um contêiner que ocupa toda a altura da janela do navegador, e o Flexbox é usado para alinhar o conteúdo no centro verticalmente.
